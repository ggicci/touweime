# Project Setup

## Bootstrap NextJS

https://nextjs.org/docs/getting-started

```bash
# bootstrap nextjs
yarn create next-app --typescript
```

## Install MUI

https://mui.com/getting-started/installation/

```bash
yarn add @mui/material @emotion/react @emotion/styled
```
